# 完整的 OWC OpenFOAM 构建流程补充说明

本文件夹已为您生成了基础的模板。以下是这套模板的几何设计依据与生成流程建议。

## 几何与环境设计

基于教授您提供的尺寸：
*   **长**: 2m（这是一个非常典型的波浪水槽缩小比例实验尺寸，或者造波验证水槽尺寸）
*   **深**: 水深 0.8m。为了让水面以上的空气室、以及产生的波峰能够容纳，总体模拟空间的高度设定为了 **1.2m**。
*   **OWC装置布置**：我在为您生成的网格打孔方案中，暂定将装置布置在中间靠后的位置 ($X=1.0m$ 处)。

## 网格与模型生成策略

如果您想在纯 OpenFOAM (不使用外部如 ICEM, Salome, Pointwise 这类画图软件) 里快速生成水槽和包含厚度墙体的模型，可以使用**“先建立大盒子，再抠出墙缝”**的方法：

1.  **第一步 (`blockMesh`)**：生成整个流域的背景粗网格（$2m \times 1.2m$）。它将只有 `inlet`, `outlet`, `atmosphere`, 和 `fixedWalls`。
2.  **第二步 (`topoSet` + `subsetMesh`)**：将前墙和后墙对应的“实体部分”选定为网格集合（我们在 `topoSetDict` 预留了位置），然后把这排网格从流域中删掉。
3.  **第三步 (重命名边界)**：删掉内部网格后，系统会产生内部暴露出来的表面（这就是您的 OWC 壁面了！）。您需要将前墙表面命名为 `frontWall`，后墙命名为 `backWall`。然后使用前面那份 `pointDisplacement` 来控制 `frontWall` 运动。
4.  （对于顶部小孔）：您可以在前墙和后墙之间靠近顶部的区域（比如 $Y=1.1m$ 处）再抠除一层网格作为挡板，或者使用无厚度挡板（Baffle）结合多孔介质面（porous jump）截留出阻尼特性。

## 波浪设置 (`constant/waveProperties`)

我在 `constant/waveProperties` 给您加入了一个 $H=0.06m$ (波高6cm)，周期 $T=1.5s$ 的默认波浪。
*   **水深 `seaLevel`**：已精确帮您设置为 0.8m。
*   **造波/消波区 (`relaxationZone`)**：我在左侧设置了 $0 \sim 0.5m$ 的造波松弛区，并在右侧 $1.5 \sim 2.0m$ 设置了消波区，以消除前墙左右晃动打出的非物理反射波。您的前墙可放在中间 $0.5 \sim 1.5m$ 的“完全求解区（working section）”内做变形实验。
